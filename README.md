# 🕵️ Device ID Logger - Weaponized Version

**⚠️ DISCLAIMER: Ini cuma buat edukasi dan testing keamanan! Jangan dipake buat hal-hal jahat, karena gue gak tanggung jawab sama dampaknya. Lo udah diperingatin!**

## 📌 Deskripsi
Sebuah tools sederhana berbasis HTML/JS untuk:
- Menampilkan animasi loading keren biar user betah nunggu.
- Mengambil **Device ID** unik dari perangkat pengunjung (pakai fingerprinting teknik).
- Redirect paksa user ke URL tujuan.
- Menyimpan semua data device ID di `localStorage` dan bisa dilihat lewat admin panel.

## 🚀 Fitur
- Animasi loading futuristik ala hacker.
- Fingerprinting: screen, userAgent, WebGL, canvas, timezone, dll.
- Redirect otomatis setelah ID terkumpul.
- Admin panel buat lihat, refresh, export CSV, dan hapus data.
